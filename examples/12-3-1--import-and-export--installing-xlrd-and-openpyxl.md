#### Installing the xlrd and openpyxl libraries in an Anaconda environment

```unix
$ conda info --envs
conda environments:
...
base * /opt/anaconda3
pandas_in_action /opt/anaconda3/envs/pandas_in_action
```

```unix
$ conda activate pandas_in_action
(pandas_in_action) $ conda install xlrd openpyxl
```
